História da web

volta

A história da World Wide Web, comumente chamada de Web, remonta ao final da década de 1980 e início da década de 1990. Foi criada por Tim Berners-Lee, um cientista da computação britânico, enquanto trabalhava no CERN (Organização Europeia para a Pesquisa Nuclear) na Suíça. A Web foi concebida como uma maneira de compartilhar informações entre cientistas em diferentes partes do mundo.

Em 1991, Berners-Lee desenvolveu o primeiro navegador web chamado "WorldWideWeb" e o primeiro servidor web, tornando possível a criação e visualização de páginas da web. Em 1993, foi lançado o primeiro navegador amplamente adotado, o Mosaic, que popularizou a Web. Posteriormente, surgiram outros navegadores como o Netscape Navigator e o Internet Explorer.

A Web evoluiu rapidamente, passando de simples documentos de texto para incluir gráficos, multimídia, interatividade e aplicativos web complexos. Hoje, a Web desempenha um papel central na vida cotidiana das pessoas, na disseminação de informações e no comércio eletrônico.

O que são Clients?

volta

Clientes, na contextura da Web, são os programas ou dispositivos que acessam e exibem conteúdo da internet. Eles desempenham o papel de solicitar informações de servidores web e interpretar as respostas para exibi-las aos usuários. Os clientes mais comuns são os navegadores web, como o Google Chrome, Mozilla Firefox, Microsoft Edge e outros, que permitem que os usuários acessem e interajam com sites da Web.

Além dos navegadores, outros tipos de clientes incluem aplicativos de e-mail, leitores de feeds RSS, aplicativos de redes sociais e muitos outros que se conectam a servidores web para obter dados ou serviços.

O que são Servers?

volta

Servidores são computadores ou sistemas de software que fornecem serviços ou conteúdo para clientes na Web. Eles desempenham o papel de receber solicitações de clientes, processá-las e enviar as respostas apropriadas de volta aos clientes. Os servidores web são uma categoria comum de servidores na Web e são responsáveis por hospedar sites e aplicativos web.

Um servidor web típico executa software especializado, como o Apache, Nginx ou Microsoft IIS, que lida com solicitações HTTP (Hypertext Transfer Protocol) enviadas pelos clientes. Essas solicitações podem ser para páginas da web, recursos estáticos (como imagens, CSS e JavaScript) ou serviços web dinâmicos que geram conteúdo personalizado com base nos pedidos dos clientes.

Sobre Linguagens de Programação Web

volta

Linguagens de programação web são usadas para desenvolver aplicativos e sites da Web. Elas permitem que os desenvolvedores criem interfaces de usuário, lógica de negócios e interações interativas na Web. Alguns exemplos de linguagens de programação web amplamente usadas incluem:

  1. HTML (Hypertext Markup Language):HTML é a linguagem básica para criar a estrutura de uma página da Web. Ela define elementos como cabeçalhos, parágrafos, links e formulários.
  2. CSS (Cascading Style Sheets): CSS é usado para estilizar e formatar páginas da Web. Ele permite que os desenvolvedores controlem a aparência visual de elementos HTML.
  3. JavaScript: JavaScript é uma linguagem de programação de alto nível que é executada no navegador do cliente. Ela é usada para adicionar interatividade a sites, criar aplicativos da Web e realizar tarefas de processamento no lado do cliente.
  4. PHP, Python, Ruby e outras linguagens de servidor:Essas linguagens são usadas no lado do servidor para processar solicitações dos clientes, acessar bancos de dados, realizar cálculos e gerar dinamicamente páginas da Web.
  5. SQL (Structured Query Language): SQL é usado para gerenciar bancos de dados e realizar operações de consulta e manipulação de dados em aplicativos web que dependem de armazenamento de informações.
  6. Frameworks web: Além das linguagens individuais, existem frameworks web como Ruby on Rails, Django (Python), Express.js (JavaScript), que oferecem ferramentas e estruturas para acelerar o desenvolvimento web, fornecendo soluções prontas para tarefas comuns.

Essas linguagens e tecnologias são usadas em combinação para criar sites e aplicativos da Web modernos e funcionais que atendem às necessidades dos usuários e das empresas online. A escolha da linguagem de programação depende dos requisitos específicos de cada projeto e das preferências dos desenvolvedores.

Aprendendo HTML na prática

Ferramentas Utilizadas

volta

No desenvolvimento web, uma série de ferramentas são utilizadas para criar, testar e depurar páginas e aplicativos da web. Algumas das ferramentas mais comuns incluem:

  • Navegadores Web: Como o Google Chrome, Mozilla Firefox e Microsoft Edge, que são usados para visualizar e testar páginas da web.
  • Ambientes de Desenvolvimento Integrado (IDEs): Ferramentas como Visual Studio Code, Sublime Text, e JetBrains WebStorm oferecem funcionalidades avançadas de edição de código, depuração e integração com frameworks web.
  • Editores de Código: Editores simples como Notepad (Windows) ou Visual Studio Code são amplamente utilizados para escrever código HTML, CSS e JavaScript.
  • Ferramentas de Inspeção de Página: Os navegadores possuem recursos de inspeção de página, como o Inspetor de Elementos, que permitem examinar o código e os estilos de uma página, bem como depurar problemas.

Usando o Inspetor de Elementos

volta

O Inspetor de Elementos é uma ferramenta embutida nos navegadores que permite examinar e modificar a estrutura e o estilo de uma página da web. Você pode abrir o Inspetor de Elementos clicando com o botão direito do mouse em qualquer elemento de uma página e selecionando a opção "Inspecionar" ou pressionando a tecla F12 ou Ctrl+Shift+I, dependendo do navegador.

No Inspetor de Elementos, você pode visualizar o código HTML, as folhas de estilo CSS e depurar JavaScript. Ele é especialmente útil para identificar problemas de layout, depurar erros de código e entender como uma página é construída.

Estrutura Básica do HTML

volta

O HTML (Hypertext Markup Language) é a linguagem de marcação usada para criar a estrutura de uma página da web. Uma página HTML básica segue uma estrutura simples:

  • <!DOCTYPE html>: Declara o tipo de documento como HTML5.
  • <html>>: O elemento raiz que engloba todo o conteúdo da página.
  • <head>: Contém informações sobre a página, como o título exibido na barra de título do navegador.
  • <title>: Define o título da página.
  • <body>: Contém o conteúdo visível da página, como texto, imagens e links.

Falando sobre Tags

volta

As tags HTML são elementos que definem a estrutura e o conteúdo de uma página. As tags são escritas entre colchetes angulares < >. Existem tags de abertura e tags de fechamento, que envolvem o conteúdo.

Por exemplo:

  • <p>: Define um parágrafo.
  • <h1> a <h6>: Define cabeçalhos de diferentes níveis.
  • <a>: Define um link.

Atributos Básicos

volta

As tags podem ter atributos que fornecem informações adicionais sobre a tag. Os atributos são especificados na tag de abertura e têm um nome e um valor. Por exemplo:

<a href="https://www.exemplo.com">Visitar Exemplo</a>

No exemplo acima, href é o atributo que define o URL de destino do link.

Executando uma Página HTML

volta

Uma página HTML é executada em um navegador da web. Você pode criar um arquivo HTML em um editor de código e salvá-lo com a extensão ".html". Em seguida, basta abrir esse arquivo em um navegador para visualizar a página.

Textos

volta

Para adicionar texto a uma página HTML, você pode usar a tag <p> para parágrafos, <h1> a <h6> para cabeçalhos e tags de texto inline como <span> e <em>.

Listas Ordenadas e Não Ordenadas

volta

Você pode criar listas ordenadas usando <ol> e <li> para itens da lista numerados e listas não ordenadas usando <ul> e <li>para itens da lista com marcadores.

Exemplo de lista ordenada:

<ol>

<li>Item 1</li>

<li>Item 2</li>

<li>Item 3</li>

</ol>

Exemplo de lista não ordenada:

<ul>

<li>Item A</li>

<li>Item B</li>

<li>Item C</li>

</ul>

Links

volta

Os links são criados usando a tag <a> O atributo href especifica o URL de destino do link. Por exemplo:

<a href="https://github.com/digitalinnovationone/roadmaps">Visite a roadmaps</a>

Esses são conceitos e elementos básicos do desenvolvimento web com HTML. À medida que você se aprofunda no desenvolvimento web, você aprenderá mais sobre estilos CSS, interatividade JavaScript e outras tecnologias web avançadas.
